Install

Install Stable version of nodejs

Install Angular/CLI

npm install @angular/cli@6.2.4 -g

Create new angular app

ng new app-name ng new app-name style --scss

Create new angular module

ng g m module-name

Run angular app

ng s ng serve

Create new angular component

ng g c component-name ng g c component-name --export

Get help about angular cli commands

ng --help ng g --help ng g c --help
